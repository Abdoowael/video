## نظرة عامة (Overview)
هذا المستودع يحتوي على الواجهات الترويجية (Reels) الخاصة بمشروع التخرج **Stomach Support** لعام 2026، من كلية الحاسبات والمعلومات. 
المشروع عبارة عن نظام خبير ذكي مبني بتقنيات الذكاء الاصطناعي لتحليل أمراض الجهاز الهضمي وتقديم توصيات طبية بدقة وسرعة.

## المميزات (Features)
- 🎨 **تصميم عصري:** واجهات جذابة مصممة لتناسب مقاسات الجوال (Reel format - 9:16).
- 🚀 **تأثيرات حركية (Animations):** تأثيرات بصرية متقدمة باستخدام CSS Animations و JavaScript.
- 💻 **تقنيات الويب:** مبني باستخدام HTML5, CSS3, Vanilla JavaScript.
- ⚡ **Tailwind CSS:** استخدام إطار عمل Tailwind CSS لتسريع وتسهيل عملية التصميم المتجاوب.

## محتويات المستودع (Repository Contents)
- `index.html`: العرض الترويجي الأساسي للنظام الخبير.
- `project_overview_reel.html`: عرض شامل لفكرة المشروع وأهدافه.
- `tech_reel.html`: عرض يركز على الجانب التقني والبرمجي للمشروع.
- `تمام222 (2).pdf`: ملف يشرح الفكرة والأهداف والجمهور المستهدف.
- `video_2026-06-23_20-55-06.mp4`: فيديو توضيحي للعرض.

## كيفية التشغيل (How to Run)
المشروع لا يحتاج إلى أي خوادم أو أدوات معقدة لتشغيله:
1. قم بتحميل أو استنساخ (Clone) هذا المستودع.
2. افتح أي ملف من ملفات الـ HTML (مثل `index.html`) مباشرة عبر أي متصفح ويب (يفضل Google Chrome أو Edge).
3. اضغط على زر "بدء التشغيل" لمشاهدة العرض الحركي.
